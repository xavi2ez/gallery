# Gallery — Hobby Portfolio

Welcome to "Gallery", a lightweight hobby portfolio website that showcases my personal projects, experiments, and creative work. This repository contains the source code and assets for the site.

## About

This project is a small portfolio built as a hobby to display web projects, artwork, and other creative experiments. It's designed to be simple, easy to update, and focused on visual presentation.

## Features

- Responsive gallery layout
- Project pages with descriptions, links, and screenshots
- Simple, minimal styling for fast load times
- Easy to add or remove portfolio items

## Tech stack

- HTML, CSS, JavaScript
- (Optional) Frameworks or bundlers used in this repo

## Running locally

1. Clone the repo:

   git clone https://github.com/xavi2ez/gallery.git
2. Open index.html in your browser, or serve the folder with a static server:

   npx http-server .

## Adding new projects

- Add a new folder for each project in the `projects/` directory (or wherever your project entries live).
- Include a thumbnail (thumbnail.jpg/png) and an index.html or markdown file with the description and links.
- Update the gallery index to reference the new project.
